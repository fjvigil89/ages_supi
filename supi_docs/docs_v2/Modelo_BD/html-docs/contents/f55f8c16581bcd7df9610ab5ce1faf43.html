ESTUDIOS REALIZADOS

ERDEntity
SUPI+ :: MODELO MEDICIONES :: ESTUDIOS REALIZADOS

Description

Para hacer la auditoría el auditor deberá: 1- Hacer la respuesta de su QUIZ 2- El sistema le mostrará el estudio que debe responder con la modalidad de ingreso que se debe y en ella se dará un valor a como lo detecte el auditor. Puede ser una palabra como QUIEBRE o un número como EL PRECIO o una foto para ver el estante vacío. A esto se podrá añadir la foto del detalle que se quiere auditar.

Siempre deberá tomarse la GEO del auditor para tener de base el dato de si el auditor está o no in-situ donde dice estar
