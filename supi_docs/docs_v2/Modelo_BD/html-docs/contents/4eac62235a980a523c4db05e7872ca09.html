MODELO MEDICIONES

ERDDataModel
SUPI+ :: MODELO MEDICIONES

Description

Es el modelo que establece la base para realizar en sí la auditoría planificada. En él se enlazan los auditores y dan fe de las visitas realizadas a cada sala y lo que obtienen en cada estudio. Antes del auditor responder debe enfrentar una entrevista a través de una consulta para que el sistema pueda avalar su capacidad para hacer dicha auditoría. En esta encuenta se le hacen 3 preguntas al azar de lo que va a hacer y se le dan 3 psosibles respuestas para cada una. Una verdadera y dos falsas. Si el auditor responde mal se le baja el rating y no se le acredita correcto su auditoría. Luego pasa a hacer las mediciones correspondientes a cada producto o mueble del estudio
