SUPI+ :: MODELO PLANOGRAMA

Description

Este modelo es para lograr organizar toda la información correspondiente a los planogramas. O sea es el mapa de las zonas existentes para auditar. Se sube por una especialista de CADEM.

Establece el tiempo en que un producto debe estar en determinado lugar de sala, sección y área. También contiene los valores normales de precio o de otra unidad que se le quiera medir a dicho producto
